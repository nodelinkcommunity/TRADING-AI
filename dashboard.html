Cong Cu Quet & Phat Hien
On-chain Data (Truc Tiep Tu Blockchain)
Aave V3 Pool Contract
Smart Contract Query
getUserAccountData() -> Health Factor cua moi borrower. getReservesList() -> Tat ca tokens duoc ho tro. Quet truc tiep, khong phu thuoc API ben ngoai.
DEX Factory Events
Event Listener
Nghe PairCreated (V2) va PoolCreated (V3) event. Phat hien pool/token moi trong < 1 block. Ho tro tat ca DEX co factory contract.
Pool Reserve Query
Smart Contract Query
Doc truc tiep reserves cua pool de tinh gia chinh xac. getAmountsOut (V2) va quoteExactInput (V3). Khong can oracle ben ngoai.
Chainlink Oracle Feed
Price Oracle
latestRoundData() -> Gia tu Chainlink. So sanh voi gia DEX de tim oracle lag. getRoundData() -> Lich su cap nhat de du doan timing.
Mempool Listener
Pending TX Monitor
pendingTransactions subscription. Thay giao dich lon TRUOC khi chung duoc mine. Dung cho JIT Liquidity va front-run protection.
Block Event Scanner
Historical Data
queryFilter() de scan events cu. Tim borrowers tu Transfer events cua debt tokens. Discovery module cho Liquidation Bot.
